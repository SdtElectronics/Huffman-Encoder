# Build from Source
## Linux 
### Requirement
* gcc 10.0.0 or above

* GNU Make 4.0 or above

### Build
```
make
```
The resultant binary will be under ../out/.

### Clean
```
make clean
```

## Windows
### Requirement
* Visual Studio 2017 or above

### Build
Open vsProj.sln in Visual Studio.
